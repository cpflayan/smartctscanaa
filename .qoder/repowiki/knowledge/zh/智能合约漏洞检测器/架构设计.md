- **入口层**: `main.py` 作为 CLI 核心，使用 `click` 框架编排扫描、获取和报告生成流程。
- **数据获取层**: `fetchers/` 模块通过 `MultiChainFetcher` 抽象多链（Ethereum, BSC等）源码获取逻辑，支持本地文件与链上地址输入。
- **扫描引擎层**: `scanners/` 采用策略模式，定义 `BaseScanner` 抽象基类，具体实现包括轻量级正则匹配 (`PatternScanner`)、静态分析 (`SlitherScanner`) 和符号执行 (`MythrilScanner`)，支持并行执行。
- **AI 分析层**: `analyzer/` 模块集成 LLM（支持 OpenAI/Azure/Ollama），对扫描结果进行二次深度分析、误报过滤及修复建议生成。
- **报告层**: `reports/` 模块负责将结构化发现（Findings）序列化为 JSON 和 Markdown 格式的安全审计报告。
- **配置管理**: 通过 `config/settings.yaml` 集中管理扫描器开关、LLM 参数及多链 API 密钥。
